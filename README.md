# CetStudentBook

## Assignment: Books CRUD

### What I implemented
Bu projede **Book** entity’si için ASP.NET Core MVC ve Entity Framework Core kullanılarak
tam CRUD (Create, Read, Update, Delete) işlemleri gerçekleştirdim.

Gerçekleştirilen işlemler:
- Kitapları listeleme
- Yeni kitap ekleme
- Mevcut kitabı düzenleme
- Kitap silme (onay ekranı ile)

Tüm işlemler veritabanı ile uçtan uca (end-to-end) çalışmaktadır.
